# Deep Learning Project: CNN + RNN + GAN

## Overview
This project implements CNN, RNN (LSTM/GRU), and GAN models using PyTorch.

## Structure
cnn/ - CNN models  
rnn/ - RNN models  
gan/ - GAN  
outputs/ - results  

## Run
pip install -r requirements.txt

python cnn/cnn_custom.py
python rnn/rnn_lstm_gru.py
python gan/gan_fashion.py
